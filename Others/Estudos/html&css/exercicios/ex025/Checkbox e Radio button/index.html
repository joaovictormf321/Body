Checkbox e Radio button

Os Checkbox e Radio button são funções de formularios para fazer marcações, um exemplo onde um dos dois é para escolher o sexo (Masculino ou Feminino), tambem pode ser usado para escolher que conteudo voce quer receber.

  • checkbox: Cria um campo de marcação.
  • radio: Cria um campo parecido com o Checkbox, porem a diferença é que o Radio é um tipo de marcação que pode ter apenas um marcado.
  • obs: Para que seja possivel ter apenas um marcado, todos os outros Radio precisa ter o mesmo (name="").

Tambem existe outros atributos e observações para ser ditas.

  • checked: Faz que und dos campos Radio ou Checkbox carregue marcado.

Observações

  • Obs: Não é possivel colocar Checked em Radio que tenha um (Name) igual de outro Radio.
  • Obs: Todo Radio que estiver dentro do mesmo grupo tem que ter o mesmo (Name) sempre.
  • Obs: È sempre bom colocar Value dentro dos Radio, para poder dar nome a eles.
