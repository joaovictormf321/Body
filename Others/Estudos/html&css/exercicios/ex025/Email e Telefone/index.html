Comunicação

Email e Telefone

Esses tipo de campos que eu aprendi hoje são usados para formularios de informação de contato, como E-mail e Telefone.

  • email: Usado para colocar E-mail, a vantagem de usar esse tipo de canpo, é que fica obrigatorio o uso de @, assim mais facil para a verificação depois no PHP ou JavaScript.
  • tel: Usado para colocar numero de telefone. Obs: No tel, é possivel usar espreçõe regulares(RegEx).

Fieldset e Legend

Esses duas novas tags para formularios são usadas para fazer agrupamento de campos dentro da tag Form.

  • fiedset: O Fieldser é usado para fazer um agrupamento dos campos do formularios.
  • legend: O Legend é usado para dar uma legenda ao agrupamento.

Obs: A tag Fieldset so funciona se ela estiver dentro da tag Form, e a tag Legend so funciona se estiver dentro da tag Fieldset.
