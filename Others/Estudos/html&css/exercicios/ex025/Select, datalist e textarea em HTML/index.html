Select, datalist e textarea

Tags para formularios, Select, datalist e textarea tags de formularios sem Input.

  • select: Representa um controle que apresenta um menu de opções.
  • datalist: Usado para criar uma associaçaõ entre o datalist e o input.
  • textarea: Cria um capo para colocar uma contidade maior de texto.
