Metodos Get e Post

Existem dois metodos de envios de formularios, o primeiro metodo é o que já vem como padrão quando nos cria um formulario, esse metodo é o (GET), o metodo GEt deixa visivel tudo qui foi enviado pelo formulario na URL do site

Já o metodo POST, não deixa tudo enviado do formulario na URL. Porem, ainda é possivel encotrar tudo enviado, entrando no (Especionar elemento), por isso, não aumenta a segurança do site, por isso tera que estudar outras linguagens de programação.

Quando usa GET e quando uar POST?

GET: O metodo Get pode ser usado quando as informações do formulario não for muito inportante,Exemplo: Quando o assunto for nome, persquisa e etc. E tambem o metodo Get, deixa uma URL para vc poder copiar, então se voce quiser mandar uma pesquisa para uma pessoa, è so copiar a URL da pesquisa. Porem, o metodo Get so pode enviar ate 3000 mil BIT e nem enviar fotos ou documentos

POST: O metodo Post já e usado para envios de coisas mais restritas como senha, email, numero de cartão de cretido, e tambem para envios de arquivos, o diferencial do Get, é que o Post pode enviar formularios com mais de 3000 mil BIT.
