Novos atributos

Um formulário HTML é feito de um ou mais widgets. Esses widgets podem ser campos de texto (linha única ou de várias linhas), caixas de seleção, botões, checkboxes ou radio buttons. A maior parte do tempo, estes elementos são emparelhados com uma legenda que descreve o seu objetivo

Atributos para formularios

Atributos Usados dentros dos formularios(input):

  • required - Deixa o campo do formulario obrigatorio para preenchimento.

  • minlength - Define a quantidade minima de caracteres do campo de formulario.

  • maxlength - Define a quantidade maxima de caracteres do campo de formulario.

  • size - Defone o tamanho da caixa do campo do formulario de acordo com o tando de caracteres.
  • placeholder - Tudo que escrever dentro desta atribuição, vai ficar dentro do campo do formulario, porem não vai atrapalhar preencher o campo.

Autocomplete
