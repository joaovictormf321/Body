Elementos color, range e file em HTML

Aprendendo novos campos de formularios, essa noite aprendi a fazer coampo de cor, range, e de arquivos em HTML.

Color, range e file

Campos Color, range e file para formularios

  • Color: Adiciona um campo de cor, usado como seletor de cor.
  • Range: Adiciona um controle para inserir um numero cujo o valor não seja importante(Como um controle deslizante).
  • File: Adicona um campo de seleção de aquivos e um botão "Procurar" para uploads de arquivos.

Atributos para Range

Usar o range sem range, acaba que pode deixar ele quase inutio, porem, existem atributos para melhorar o uso dele.

  • Max: Especifica o valor maximo permitido.
  • Min:Especifica o valor minimo permitido.
  • Step: Especifica os intervalos de numeros legal.
  • Value: Especifica o valor padrão.

Obs: O atributo Value poder ser usado no Range como tambem no Color, no color é usado para definir uma cor padrão.
